# Willow Child Development Center

A website for Willow Child Development & Rehabilitation Center, supporting children with speech, developmental, and learning needs.

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Framer Motion
- React Router DOM

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Install dependencies
npm install

# Start the development server
npm run dev
```

### Build

```sh
# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

- `/src/components` - React components
- `/src/pages` - Page components
- `/src/components/ui` - UI components (shadcn)
- `/src/components/layout` - Layout components (Navigation, Footer)
- `/src/components/home` - Home page sections
